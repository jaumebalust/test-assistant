# Test Assistant 

Because, let's face it. You never write enough tests...

Test assistant to the rescue.

## Installation
Run in your terminal:
```
npm i -g @jaumebalust/test-assistant
```
Add the ENV variable to your .bashrc or .zshrc file:
``` 
OPENAI_API_KEY
```
## Usage
Run in your terminal:
```
test-assistant
```
